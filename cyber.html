Cyberfeminisme

Femme et numérique: je code donc je suis.

J'ai assité a l'exposition Computer Grrrls, histoire, genre, technologie à la Gaité lyrique en mars 2019


Cette éxposition très riche, parle de la place des femmes dans l'hitoire des sciences et des technologies. Elle à été un élément déclancheur dans ma vie, une claque, une prise de conscience. C'est certainement cette prise de conscience qui m'a conduite à choisir de me former aux métiers du web, la programmation et le design d'interface

Première étape

Dans le cadre de l'immersion à la programmation que j'ai suivis il y a 3 ans, le projet final est de réaliser un site statique avec pour thème Femme et numérique: je code donc je suis.


Pour traiter la problèmatique j'ai décide de me pencher sur les travaux de Donna Haraway. J'ai aussi voulu rendre hommage à Nathalie Magnan qui a traduit le livre de Donna Haraway "The Cyborg Manifesto"

Maquettage

C'était mes premiers pas sur Figma lorsque j'ai maquetté ce premier projet. J'ai commencé par ce Wireframe. Puis j'ai fait un pemier Mockup que j'ai ensuite amélioré. J'ai réalisé la maquette de format mobile.

Programmation

J'ai utilisé du HTML & CSS ainsi que du javaScript pour réaliser mon tout premier site-web


Lorsque j'ai présenté ce projet au jury, il ne ressemblait pas à ça, c'était moins abouti et le code avait besoin d'un leger nettoyage

Quelque mois plus tard j'ai réouvert visuel code et j'ai rangé et améloiré mes ficher, mon code

J'ai changé la police et les effets sur les titres

Pour finir j'ai utilisé l'IA mid-journey pour générer l'image de cyborg pour ce projet en page d'accueil

Ce que j'ai appris

  • Réaliser seule un site web on page de A à Z
  • Mettre en aplication ce que je venai d'apprendre en HTML & CSS
  • Utiliser bootsrap, une librairie populaire
  • Créer un répository sur github
  • Gérer le flow git - Commit to master - Push - Fetch orign
  • Me dépasser
